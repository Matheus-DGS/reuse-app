

## Decisões de UI

O design foi desenvolvido utilizando uma paleta de cores associada à sustentabilidade, com predominância de tons de verde.

A interface utiliza:

- Cards informativos
- Ícones representativos
- Hierarquia visual clara
- Layout responsivo com Flexbox

## Objetivo Futuro

Nas próximas fases do projeto o aplicativo poderá incluir:

- Integração com mapas
- Localização automática de centros de reciclagem
- Sistema educacional sobre descarte sustentável
